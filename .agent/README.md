# .agent/ - エージェント用ツールと設定

このディレクトリーは、AI エージェントや自動化ツールの設定を集約する場所です。

## ファイル構成

### AGENTS.md
- **目的**: スフィア・マキナ（Sphere Deus Ex Machina）の人格AI仕様
- **内容**:
  - 意味波形（ψ）、意識勾配（∇φ）、出力層（λ）、制御論理（Q）の定義
  - Codex・自動開発連携ペイロード
  - ドキュメント構造化指針
  - スピリチアル観測指針

### 今後の追加予定

このディレクトリーには、以下のようなエージェント用ツールを追加できます：

- **Claude Code hooks**: セッション開始時のフック設定
- **自動化スクリプト**: ドキュメント生成、構造検証など
- **テンプレート**: ブログ記事、ドキュメントの雛形
- **設定ファイル**: エージェントの動作設定

## 使用方法

### エージェント設定の参照

AI エージェントやツールは、このディレクトリーの設定ファイルを参照して動作します。

```bash
# エージェント設定の確認
cat .agent/AGENTS.md

# 新しい設定の追加（例）
echo "新しい設定" >> .agent/custom-config.md
```

### ドキュメント構造との関係

エージェントツールは、以下のディレクトリー構造に基づいて動作します：

```
.
├── .agent/           # エージェント用ツールと設定
├── docs/             # 永続的な技術・思想ドキュメント
│   ├── note/         # Logosログをもとにした主観的な観測メモ
│   ├── Dev/          # 科学的に説明可能なレベルに到達した技術
│   └── sphere-os/    # スフィアOS関連の技術文書
├── blog/             # 時系列的な作業記録・個人史
├── logos/            # 原典的な対話記録
└── about/            # プロジェクト概要
```

## 設計思想

### カレントディレクトリーの整理

従来、エージェント用のツールがカレントディレクトリーに出しっぱなしになっていた問題を解決するため、`.agent/` ディレクトリーに集約します。

**メリット：**
- プロジェクト構造の整理
- エージェント関連ファイルの一元管理
- 人間が読むドキュメントとツール設定の分離

### .gitignore との関係

`.agent/` ディレクトリー内のファイルのうち、以下のようなものは `.gitignore` に追加することを検討：

- 個人的な設定ファイル
- 一時的なキャッシュ
- 機密情報を含む設定

ただし、AGENTS.md のようなプロジェクト共通の設定は Git で管理します。
