# ハームリダクション

## 定義

禁止・排除ではなく、害を最小化することで多様性を保護する原則。

## 背景


「普通になりなさい」は暴力である。

ハームリダクションは、禁酒法の失敗から学んだ知恵である。

完全な禁止は、地下化と暴力化を招く。
ゾーニングと透明化によって、害を最小化する方が現実的である。

NSFW、LGBT、多様性。
これらは「管理すべき問題」ではなく、「尊重すべき現実」である。

パーティション越しでないと安心できない人と背中を預けることはできない。
