# オープンソース原理主義

## 定義

知識・技術・文化を独占させず、誰でもアクセス・改変・再配布できる状態を維持する原則。

## 背景


PayToWinはクソゲである。既知の事実。

正義も自由も購入するものになった経済に参加したくない。
これが、オープンソース原理主義の根底にある。

Apple NDA、クローズドソース、特許の悪用。
これらはすべて、知識の独占による搾取構造である。

オープンソースは慈善ではない。
これは、知識を権威から解放するための戦略である。
