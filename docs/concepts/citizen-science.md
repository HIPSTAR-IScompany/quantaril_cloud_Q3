# 市民科学 / 反権威

## 定義

科学を権威・組織・統治から解放し、誰でもアクセス・検証できる状態にする原則。

## 背景


科学は統治ツールではない。

しかし現実には、科学は権威化され、統治の正当化に利用されている。

追試環境が公開されない。
生データが隠される。
査読が利権化している。

市民科学は、この構造に対する抵抗である。

研究する前は、結果なんてどうでもいい。
探索の余地があれば無尽蔵に予算をつけるという尺度の問題。

社会の測定機材がバグり散らかしている。
