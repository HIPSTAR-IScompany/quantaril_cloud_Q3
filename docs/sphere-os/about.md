# 設計哲学と情報子工学について

これは、ゲーミフィケーション、IT構文、SF創作的視点を用いた、
あくまで“設計哲学”と“人生哲学”としての一つの学説です。

## helloworld

この設計哲学を学ぶための最初のことば「helloworld」
壮大に宇宙の真理みたいな絶対的なものと捉えないでください。

そもそもこのスフィアOSという開発環境やゲーミング宇宙論はただの哲学的な概念です。

量子コンピューターでなくともクロスプラットフォームで幅広く使えるもので
データーサイエンスの新しい手法や評価制度の考え方などフレームワークの側面が強いです。

そもそも可搬的でフラクタルで、波動性があることが条件なので
どんな、小さなことから、大きなことまであくまで、考え方として存在してます。

例えばSNSに呟く愚痴と自慢の一言の共感や学びから、国政や企業の人事など
フラクタルにカプセライズしてメッセージ思考化して統合することで、
インクルーシブと幸福追求を両立しようという哲学です。

## 情報子科学と工学

情報に波動性と量子性を持たせるこことで情報工学・データーサイエンスに熱力学など、
物理学的普遍性を付帯することを目的とします。

物理学的普遍性により現実に即して実用性や高度なセンシングに最適化する
情報子科学という新しい学派の提唱です。

量子演算やAIなど高次元ベクトルと波動性に代表されるベクトルDB時代の情報工学・データーサイエンスの可搬性と
非破壊性を両立するアプローチ法を提唱することで、RDB時代の静的なデータ工学から動的な現実に即したデーター工学と
しとらえる試みが情報子科学です。

さらにその動的なデーター加工と量子テクノロジーや高エネルギー物理学などと合わせて
新たな科学製品や技術を開発する手法を情報子工学と新たに学派を提唱しているに過ぎません。

## ゲーミング宇宙論

フラクタルに教育思考や研修システムやゲーム開発や社内評価制度などに利用できる、
人間の成長と環境と社会工学をフラクタルに包括しつつ、役割の多様性や生きがいと誰かを取り残さない設計哲学。

つまり成長を実感できるというエクスペリエンス基準の幸福追求哲学の一つの哲学学説であり、
成長という概念である以上、過去の様々な宗派の霊的や宗教的なものにコンフリクトしません。

会社資本の成長から、国益の追求から諸国の憲法にある幸福追求権や他の何かの追求とのコンフリクトを防ぎ
構造を設計する設計哲学や人生哲学の一つの考え方であります。

### 🧠 PvEとPvPは“神と魔王の世界観の共存モデル”

* \*\*PvE（Player vs Environment）\*\*は主に神々や天使の設計思想であり、魂を鍛えるための試練の道でした。
* 一方、\*\*PvP（Player vs Player）\*\*は魔王・鬼・戦神など、**力による覚醒を促す存在たちの設計思想**でした。

  * 日本では「軍神」「修羅神」「破壊神」として神話に登場する存在たちも、**決して邪ではなく、違う学習方式を持つ存在**です。
  * 現代で言えば、PvP格闘・FPSゲー・ぷよぷよ・スポーツ・コロシアム・エクストリームスポーツなどがこれに該当。

🔄 つまり本来、**PvEとPvPはFold空間を安全に分離して同時運用する前提設計**であり、
「争い」と「学び」は構文的に調和できるものだったのです。
